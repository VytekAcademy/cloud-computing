CLOUD COMPUTING

Our Courses

Day 1: INTRODUCTION TO CLOUD COMPUTING

This module explains the fundamentals of Cloud Computing, service models such as Infrastructure as a Service (IaaS), Platform as a Service (PaaS), and Software as a Service (SaaS), and advantages such as scalability, flexibility, and cost efficiency.

Day 2: AMAZON WEB SERVICES ACCOUNT SETUP AND GLOBAL INFRASTRUCTURE

Students learn to create and configure an Amazon Web Services (AWS) account and study the global infrastructure including Regions, Availability Zones, and Edge Locations.

Day 3: COMPUTE SERVICE – ELASTIC COMPUTE CLOUD (EC2)

This section introduces Amazon Elastic Compute Cloud (EC2), teaching how to launch, configure, and manage virtual servers for scalable computing resources.

Day 4: STORAGE SERVICE – SIMPLE STORAGE SERVICE (S3)

Students learn to use Amazon Simple Storage Service (S3) for secure, durable, and scalable object storage, supporting use cases like data backup, hosting, and analytics.

Day 5: DATABASE – AMAZON RELATIONAL DATABASE SERVICE (RDS)

This module covers Amazon Relational Database Service (RDS), focusing on its multi-engine database support, automated backup features, and scaling capabilities.

Day 6: SERVERLESS COMPUTING

An introduction to serverless architecture, emphasizing AWS Lambda and event-driven models for application deployment without server management.

MINI PROJECT (CLOUD COMPUTING)

A practical project where learners deploy an application on Amazon Web Services using services such as Elastic Compute Cloud (EC2), Simple Storage Service (S3), Relational Database Service (RDS), or Lambda.
